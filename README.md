Assignment 2 – Searching and Sorting Examples in C
--------------------------------------------------

This repository contains my **Assignment-2** for the subject **Data Structures and Algorithms (DSA)**  
under **BCA (Artificial Intelligence)** at **Invertis University, Bareilly**.  
All programs are written in **C language** and include examples for searching and sorting techniques.

--------------------------------------------------

Programs Included

| No. | Question | File Name | Description |
|:--:|:-----------------------------------|:----------------------|:----------------------------------------------|
| 1 | Linear Search Example in C | `Q1_Linear_Search.c` | Program to search an element in an array using Linear Search |
| 2 | Binary Search Example in C | `Q2_Binary_Search.c` | Program to search an element using Binary Search |
| 3 | Bubble Sort Example in C | `Q3_Bubble_Sort.c` | Program to sort elements using Bubble Sort algorithm |
| 4 | Selection Sort Example in C | `Q4_Selection_Sort.c` | Program to sort array elements using Selection Sort |
| 5 | Insertion Sort Example in C | `Q5_Insertion_Sort.c` | Program to insert and sort elements using Insertion Sort |
